# Features

Feature modules contain application capabilities that are visible in the product flow or AI architecture.

- `AiGateway`: model provider routing and model request contracts.
- `Documents`: document upload, parsing, preview data, and document providers.
- `Skills`: reusable document capabilities such as summary, risk analysis, email draft, and classification.
- `ToolGateway`: internal tool catalog and execution.
- `Mcp`: external MCP-compatible interface over selected tools.
- `Workflows`: multi-step application flows that compose skills and tools.
- `Rag`: retrieval, embeddings, vector search, and citations.
