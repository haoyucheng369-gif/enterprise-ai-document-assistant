# Frontend

React Workspace for Enterprise AI Document Assistant.

## Current Scope

- Document list
- Document detail view
- Right-side AI Assistant shell
- Citation panel placeholder
- Tool result panel placeholder
- Tailwind CSS utility styling
- Component-based workspace structure

## Structure

```text
src/
├── components/
│   ├── AssistantPanel.tsx
│   ├── CitationPanel.tsx
│   ├── DocumentNav.tsx
│   ├── DocumentPreview.tsx
│   ├── DocumentWorkspace.tsx
│   ├── ToolResultPanel.tsx
│   └── WorkspaceHeader.tsx
├── data/
│   └── workspaceData.ts
├── App.tsx
├── index.css
└── types.ts
```

## Commands

```bash
npm install
npm run dev
npm run build
npm run lint
```
